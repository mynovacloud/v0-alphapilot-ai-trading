Safety
Live trading is intentionally disabled. To enable it you must set LIVE_TRADING_ENABLED=true in your .env file and restart the server. Even then, no real-API trade execution code is wired up.

All API credentials are stored as encrypted-at-rest placeholders. Replace the placeholder columns in api_credentials_placeholder with real encrypted storage (e.g. cryptography.fernet or an OS keychain) before going live.
